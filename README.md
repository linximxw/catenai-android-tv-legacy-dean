# Catenai Android TV Legacy 4.4

Android 4.4 legacy client built from scratch for the Catenai platform.

## Current Goal

Bootstrap a Java + XML Android project that:

- targets `minSdkVersion 19`
- uses `targetSdkVersion 28`
- follows the platform guide contracts
- references the 5.0 snapshot only for behavior and API mapping
